## 参数说明
为方便跑友计算不同配速下5KM/10KM/半马/全马所用时间，开发了此工具。

* 1.你只需输入你想要计算的配速x分x秒，即可计算出全马所有公里数累计用时。

* 2.增加各种跑步模式推荐心率。
